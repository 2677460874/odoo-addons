=====================
HR Applicant Reminder
=====================

This module provides functionality to notify the followers regarding HR Recruitment Application
based on following rules:

* If today is Mon, reminder is sent for Action date that are on Wed (+48h)
* If today is Tue, reminder is sent for Action date that are on Thu (+48h)
* If today is Wed, reminder is sent for Action date that are on Fri (+48h)
* If today is Thu, reminder is sent for Action date that are on Mon (+96h)
* If today is Fri, reminder is sent for Action date that are on Tue (+96h)
